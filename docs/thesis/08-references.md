# References and Citation Guidance

> **Purpose.** This file provides (a) practical guidance on IEEE and APA formatting and
> reference management, and (b) a ready-to-use, IEEE-numbered reference list whose numbers
> `[1]`–`[21]` are used consistently across all chapters. Verify each entry's details (DOI,
> pages, access dates) before submission; bracketed placeholders mark fields to confirm.

---

## 1. IEEE citation format (recommended for this thesis)

**In-text.** Cite by bracketed number in order of first appearance: "RAG grounds generation in
retrieved evidence [7]." Reuse the same number on later mentions. Group multiple: "[9], [12]"
or a range "[1]–[3]".

**Reference list.** Numbered in citation order. Common templates:

- **Journal article.** `[n] A. Author, B. Author, and C. Author, "Title of paper," Journal
  Name, vol. X, no. Y, pp. mm–nn, Mon. Year, doi: 10.xxxx/xxxxx.`
- **Conference paper.** `[n] A. Author and B. Author, "Title," in Proc. Conf. Name, City,
  Country, Year, pp. mm–nn.`
- **Book.** `[n] A. Author, Title of Book, xth ed. City, Country: Publisher, Year.`
- **Online / docs.** `[n] Organization, "Title," Year. [Online]. Available: https://URL
  (accessed Mon. DD, Year).`

## 2. APA alternative (if your department requires it)

**In-text.** Author–date: "(Lewis et al., 2020)". **Reference list.** Alphabetical by author:
`Author, A. A., & Author, B. B. (Year). Title of work. Publisher/Journal, volume(issue),
pages. https://doi.org/...`

## 3. Reference management

- Use **Zotero** (free) with the **IEEE** citation style; the browser connector captures papers
  and DOIs automatically.
- If writing in **LaTeX**, export a `.bib` file and use `IEEEtran.bst`. If writing in **Word**,
  use the Zotero Word plug-in and insert a dynamic bibliography.
- Keep a single library so numbering stays consistent across chapters.

## 4. Reference-quality evaluation (use this checklist)

A strong reference list is **recent, peer-reviewed, primary, and relevant**:

- **Recency** — prefer 2018–2025 for AI/IR techniques; seminal classics are exceptions.
- **Venue quality** — top venues (NeurIPS, ACL, SIGIR, IEEE/ACM journals) over unreviewed blogs.
- **Primary over secondary** — cite the original paper, not a summary.
- **Balance** — mix foundational (Transformer, BM25) with current (RAG, LightGBM) and
  domain/context (Egypt/MENA labour reports).
- **Target count** — a B.Sc. thesis typically cites **25–45** references; the list below is a
  validated core to extend.
