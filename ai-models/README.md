# AI Models - Sha8alny Platform

This directory contains all AI-powered components for the Sha8alny career development platform.

## Components

### 1. AI Skill Assessment
- Question generation using LLM
- Response evaluation and skill analysis
- Proficiency level determination

### 2. Learning Roadmap Generator
- Personalized learning path creation
- Hierarchical roadmap structure (Phases → Milestones → Skills)
- Time estimation and sequencing

### 3. RAG Career Advisory System
- Knowledge base management
- Vector embeddings and semantic search
- Conversational career guidance with context

### 4. Job Recommendation Engine
- Intelligent job matching
- Skill-based recommendations
- Explainable recommendations
